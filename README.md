# GD-Internal-MacroBot (Geode C++ Edition)

Высокоточный, встроенный макрос-бот (Recorder/Player) для Geometry Dash (версия 2.2081+) на языке C++. Бот компилируется через легковесный MinGW и внедряется в игру как Geode-мод, обеспечивая Frame-Perfect точность, поддержку ClickBot и обход чит-индикаторов.

---

## 🛠 Технические особенности и фичи

1. **Внутреннее меню (GUI)**: Меню бота интегрируется прямо в интерфейс игры (в окно паузы или по горячей клавише), используя стандартные UI-компоненты Geode SDK / Cocos2d-x. Оно открывается по нажатию определённой клавиши (типа F6).
2. **Frame-Perfect запись**: Бот перехватывает оригинальные функции клика игры (`GJBaseGameLayer::handleButton`) и записывает действия строго по номерам внутренних кадров (`frameCounter`).
3. **Сохранение в JSON**: Макросы сохраняются в папку `geode/mods/your.mod.id/macros/` в классическом формате:
   ```json
   [
     {"frame": 124, "button": 1, "push": true},
     {"frame": 150, "button": 1, "push": false}
   ]
   ```
4. **Легитность для MegaHack**: Бот работает внутри экосистемы Geode, поэтому встроенный Cheat Indicator в MegaHack считает его обычной легальной модификацией и не загорается красным.
5. **Поддержка ClickBot**: Так как бот вызывает оригинальные функции нажатия кнопок движка игры, все установленные моды на звуки кликов (ClickBot) будут автоматически и синхронно озвучивать воспроизведение макроса.

---

## 📋 Инструкция по компиляции через терминал (MinGW)

Вместо тяжелой Visual Studio мы используем установленный компилятор `g++`. Сборка DLL-модификации выполняется одной командой в терминале OpenCode:

```bash
g++ -O3 -shared -std=c++20 -o GeodeMacroBot.dll src/main.cpp -luser32 -lole32
```
*(Примечание: Для полноценного связывания с Geode рекомендуется использовать утилиту `geode CLI`, встроенную в SDK).*

## 🏗 План разработки для AI-Агента

### Шаг 1: Создание структуры Geode-мода (`src/main.cpp`)
*   **Задача**: Подключить заголовочный файл Geode (`#include <Geode/Geode.hpp>`) и настроить хуки (Hooks) на функции мыши, пробела и кадра игры.
*   **Инструкция ИИ**: Напиши базовый C++ код с использованием Geode-хуков для отслеживания текущего кадра уровня через `PlayLayer::get()->m_gameState`.

### Шаг 2: Модуль Записи и Воспроизведения (Recorder/Player)
*   **Логика**: Напиши функции `recordClick(int frame, int button, bool isPress)` и `playClick(int frame)`. Для чтения/записи JSON файлов используй встроенную в Geode библиотеку `matjson`.

### Шаг 3: Внутриигровое меню
*   **Логика**: Создай кастомный слой меню (`CCLayer`), который открывается кнопкой из меню паузы уровня. Добавить кнопки: `Record`, `Play`, `Save` и выпадающий список макросов.
